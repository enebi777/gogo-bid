# GoGo Bid — Enterprise Account Settings (⚙) Implementation

## Objective

Transform the current **⚙ Settings** button on every connected account into a complete **Account Control Center**.

This is **not** an edit modal.

It is the command center for each connected account.

Examples:

* Meta Business Manager
* Google MCC
* TikTok Business Center
* Taboola Account
* Voluum Workspace
* ClickBank Account

The UI should feel comparable to:

* Stripe
* HubSpot
* Vercel
* Linear
* Notion

Use a right-side slide-over panel (preferred) or full-page drawer.
