# GoGo Bid – UI/UX & Campaign-Centric Architecture Optimization Prompt

## Objective

Redesign and refactor the entire GoGo Bid application to create an intuitive, modern, enterprise-grade user experience while preserving all existing functionality.

The current application feels feature-heavy and complex. The goal is **not to remove features**, but to reorganize, simplify, and guide users so every capability is easy to discover and understand.

The application should feel comparable to modern SaaS platforms such as HubSpot, Monday.com, ClickUp, Notion, Stripe Dashboard, Datadog, and Windsor.ai—powerful, yet approachable.

---

# Core Principle

## Everything revolves around the currently selected Campaign.

The Campaign is the central object of the entire platform.

Every module, AI engine, intelligence system, report, automation, integration, workflow, and analytics page should automatically operate on the currently selected Campaign.

Think of the Campaign as the application's "workspace."

```
Campaign
│
├── Intelligence
│     CPC Intelligence
│     CPA Intelligence
│     Offer Analyzer
│     GEO Analysis
│     Anomaly Detection
│     Benchmarks
│
├── AI Tools
│     AI Copilot
│     AI Forecasting
│     AI Simulator
│     Scaling Engine
│     Creative Studio
│     Audience Builder
│
├── Analytics
│     Budget Pacing
│     Attribution
│     Reports
│
└── Automate
      Automation
      Integrations
```

No module should require the user to repeatedly select campaigns.

Instead:

* Campaign is selected once.
* Every page automatically uses that Campaign context.
* Switching campaigns updates every module instantly.

All modules should share context.

Example:

↓

↓

↓

Reports reflect projected improvements.

↓

The platform should behave as one integrated intelligence system rather than isolated tools.

# Final Goal

Transform GoGo Bid into an enterprise-grade, AI-first, campaign-centric marketing intelligence platform where every feature is connected through the currently selected Campaign.

The interface should prioritize:

* Simplicity
* Clarity
* Readability
* Discoverability
* Guided workflows
* Context awareness
* Progressive disclosure
* AI-assisted decision making
* Consistent navigation
* Excellent onboarding
* Enterprise scalability

The final experience should make advanced marketing intelligence feel effortless, intuitive, and approachable while preserving the platform's full power for advanced users.
