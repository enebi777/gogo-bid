# Campaign Context System Enhancement – Creative Studio & Audience Builder Integration

## Objective

Enhance the GoGo Bid Campaign Context System so that **every feature in the application is automatically bound to the currently selected Campaign**. Users should never have to manually reselect a campaign when navigating between modules.

The selected Campaign acts as the application's global workspace and serves as the single source of truth for all data, AI, analytics, creative assets, audience management, and automation.

---

# Global Campaign Context

Implement a persistent global Campaign Context that is shared across the entire application.

The active Campaign should always be visible in the top navigation and remain synchronized across every page.

Example:

```text
Current Campaign

Facebook • Nutra US • Spring 2026
```

or

```text
Campaign ▼
Affiliate Max
```

Changing the active Campaign should instantly update every module without requiring a page refresh or additional user interaction.

---

# Creative Studio Integration

Creative Studio must be fully Campaign-aware.

When the user changes the active Campaign, Creative Studio should immediately load:

* Campaign creative assets
* Ad creatives
* Images
* Videos
* Headlines
* Primary text
* Descriptions
* CTAs
* Brand guidelines
* Color palettes
* Generated AI creatives
* Creative performance metrics
* Creative history
* Creative versions
* Creative recommendations
* Creative testing results

AI-generated creatives must always use the active Campaign's:

* Offer
* GEO
* Audience
* Objectives
* Brand voice
* Platform
* Historical performance
* Existing assets

No manual Campaign selection should exist inside Creative Studio.

# Performance Requirements

Changing the active Campaign should:

* Instantly update the application state.
* Refresh all dependent modules.
* Preserve unsaved work when possible.
* Avoid unnecessary reloads.
* Use cached data where appropriate.
* Synchronize background API requests efficiently.
* Deliver a seamless, real-time experience.

---

# Final Goal

The Campaign Context System must become the foundation of the GoGo Bid platform.

Every feature—including Intelligence, AI Tools, Analytics, Automation, Creative Studio, Audience Builder, Integrations, Reports, and AI assistants—must automatically operate within the currently selected Campaign.

The result should be a unified, context-aware, AI-first experience where users interact with a single Campaign workspace instead of navigating disconnected modules or repeatedly selecting Campaigns.
